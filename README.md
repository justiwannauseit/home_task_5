### Description

```
    Скрипт для поднятия selenoid в docker для Windows
    (Важно!) Все скрипты выполняются в PowerShell,
    поэтому версия вашей ОС должна быть выше Vista
```

### Install

```
    git clone https://github.com/justiwannauseit/home_task_5 
```

### Run

```
    Вы можите клонировать данный проект и запустить его в вашей IDE.
    Либо в папке /script уже сгенерирован исполняемый jar файл.
    Чтобы запустить script в консоле выполните команду:
    java -jar dockerScript.jar
    
    (Важно!) Перед использованием скрипта, убедитесь, что у вас установлен:
    1. Docker
    2. VirtualBox
    3. PowerShell
    Так же перед запуском скрипта запустите Docker Quickstart Terminal, и
    убедитесь, что все работает коректно и без ошибок. 
    Не выключайте терминал до тех пор пока скрипт не отработает.
```
